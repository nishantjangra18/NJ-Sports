# NJ Sports

Premium sports streaming platform built with Next.js, TypeScript, Tailwind CSS, Framer Motion, Lucide icons, TanStack Query, and Zustand.

## Brand Asset

This project is configured to use one official logo file everywhere:

`public/brand/nj-sports-logo.png`

Place your official NJ Sports logo at that path. The app uses the same file for the visible brand logo, favicon, app icon, splash/loading logo, browser tab icon, and PWA manifest icons.

Do not redesign, recreate, modify, replace, or generate another logo.

## Run Locally

```bash
npm install
npm run dev
```
